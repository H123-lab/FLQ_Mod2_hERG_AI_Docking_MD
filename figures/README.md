# Figures

This directory contains structural visualizations and docking analysis figures supporting the computational study of fluoroquinolone scaffold redesign.

Figures illustrate:

• Ligand structure comparison
• hERG channel binding pocket
• Docking pose overlays
• Interaction mapping
• Docking score comparison
• Key aromatic interaction residues
• Computational workflow pipeline

All figures were generated using ChimeraX, PyMOL, PLIP, and molecular docking analysis tools.
