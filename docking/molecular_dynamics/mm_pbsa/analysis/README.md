# Analysis Data

This folder contains structural stability analysis data derived from molecular dynamics simulations of the hERG channel complex.

## Files Included

• rmsd_data.csv — Root Mean Square Deviation values describing ligand stability during simulation.

• rmsf_data.csv — Root Mean Square Fluctuation values describing residue flexibility across the trajectory.

These datasets were used to generate RMSD and RMSF plots included in the supplementary figures of the manuscript.
