# Docking Data

This folder contains files related to molecular docking of fluoroquinolone analogs
against the human hERG potassium channel.

Protein structure used:
PDB ID: 5VA1

Docking software:
CB-Dock2
AutoDock Vina

Contents of this folder:

hERG_5VA1_prepared.pdb
FLQ_Mod2_docking_pose.pdb
docking_protocol.txt

These files document the docking setup and representative binding pose used
for visualization and analysis in the study.
