# Ligand Structures

This directory contains the ligand structures used in the computational workflow.

Structures were generated from SMILES strings reported in the manuscript and
converted into 3D geometries using Schrödinger LigPrep (OPLS4 force field).

Ionization states were assigned at physiological pH (7.4 ± 0.5).

Ligands included:

- moxifloxacin (reference fluoroquinolone)
- FLQ_Mod_2 (redesigned scaffold prioritized in the study)
- FLQ_Mod_5
- FLQ_Mod_6

These structures were used for:

• docking simulations  
• molecular dynamics simulations  
• MM/GBSA binding free energy calculations  

All structures are provided in SDF format for reproducibility.
